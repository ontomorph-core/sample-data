# Ontomorph sample data

Synthetic sample data for building and testing against the Ontomorph platform and the HOLON clinical knowledge API. None of this is real patient data.

## Layout

- `twins/` — synthetic digital twins (patients, health events across systems)
- `grants/` — demo grant tokens for connecting to the sample twins
- `holon/` — example concept, interaction, and mapping payloads

## Usage

Point your test keys at these fixtures while building. See the developer docs at https://developer.ontomorph.com/docs.

> Scaffold only. Populate with fixtures as they are prepared.
